## Demo

[chat](https://cryptic-brook-36573.herokuapp.com/)

## Description

This is a simple real-time chat web application based on socket.io


## Prerequisites

`nodejs ^8.11.3`

## Instalation

```sh
git clone https://github.com/ErnestZiemkowski/chat
cd chat/
npm install
```
## Run app locally

Go back to the root directory of chat

```sh
npm start
```

## Stack & Tools

- express.js
- node.js
- socket.io
